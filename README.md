# RoboSport Clone Research Pack

This folder captures what we have determined so far about building a RoboSport-faithful clone.

It is intentionally structured for use with Codex/Cursor/other coding agents. The key rule is: do not treat inferred mechanics as confirmed original mechanics.

## Recommended reading order

1. `docs/source-notes.md`
2. `docs/confirmed-mechanics.md`
3. `docs/open-questions.md`
4. `docs/resolution-rules-proposal.md`
5. `tests/original-game-test-plan.md`
6. `prompts/codex-research-prompt.md`
7. `prompts/codex-implementation-prompt.md`

## Important caveat

The exact original RoboSport conflict/collision rules are not yet confirmed. The proposed resolution model in this pack is a deterministic implementation plan suitable for a clone, but should be updated if the full technical reference or empirical testing reveals different original behavior.
